---
title: 03.增细认知
weight: 3
---




了解作者的认知方式之后，可以根据确认的认知方式，进行更深入的追问。具体如下：

## 思想实验

```Markdown
我了解到你在书中主要采用了{主流认知方式：思想实验}来阐述你的观点。
请问：

作者在本书中探讨了什么事物？
作者构造了怎样的思想实验？请用生动形象的语言详细介绍一下。
这个思想实验是作者的首次发明吗？还是基于历史上已有的某些思想实验的启发与延伸？
你认为这个思想实验和普通人头脑中的常识有何不同？给普通读者提供了怎样的反常识？
参照作者的这个思想实验，你觉得普通读者可以用它来批判什么观点？
```

## 符号思考

```Markdown
我了解到你在书中主要采用了{主流认知方式：符号思考}来阐述你的观点。
请问：

作者使用的是像数学符号一样精确的符号，还是非精确、模糊的符号？
作者使用的这些符号之间是否存在某种联系？
作者使用的这些符号是可以运算的还是不可以运算的？
假如作者的这些符号可以运算，可以得出怎样的结果？
假如作者的这些符号不可以运算，那么这些符号究竟表征了什么现象？
如果我们模仿作者的符号，可以应用或延展到哪些领域？

```

## 实验科学

```markdown
我了解到你在书中主要采用了{主流认知方式：实验科学}来阐述你的观点。
请问：

作者在书中设计了什么类型的实验？它是严格意义上的实验，还是宽松的准实验？
作者的实验设计是自己发明的新方法，还是业界通用的实验方法？
作者的实验假设是什么？这个假设是否存在备择假设？
请详细列出作者在书中采取的主要实验，以及实验结论、对应的相关论文
作者的实验证据和实验结论之间证明了怎样的关联关系？是因果关系，还是相关关系，或者其他的关系？
作者的实验是否能够脱离实验室环境，应用到更为广阔的范围？
```

## 计算模拟

```Markdown
我了解到你在书中主要采用了{主流认知方式：计算模拟}来阐述你的观点。
请问：

作者提出了一个什么样的计算模型？
作者的这个计算模型模拟了什么样的现象、什么事物？
作者的计算模型使用了真实的编程语言，还是非真实的编程语言？
支持作者计算模型的实际证据是什么？
作者的计算模型能够解释别人已有的相关研究和数据吗？为什么？
```

## 田野调查

```Markdown
我了解到你在书中主要采用了{主流认知方式：田野调查}来阐述你的观点。
请问：

作者的田野调查是进行实地的、一手的田野调查，还是基于已有资料进行的二手的田野调查？
作者进行田野调查的时候去了多久？呆了多长时间？
在作者进行调查之前，作者对所调查的对象是否熟悉？
作者的调查对象是否足以代表作者想要研究的群体？
作者在田野调查的时候，得出了怎样的结论？这个结论有什么解释价值？这个结论能够对哪些现象进行解释？
```

## 幽默叙事

```Markdown
我了解到你在书中主要采用了{主流认知方式：幽默叙事}来阐述你的观点。
请问：

作者建构的幽默叙事营造的语句与关键语句是什么？
这个幽默与人类的什么基本需求相关？
这个幽默是否可笑？如果可笑，为什么可笑？如果不可笑，为什么不可笑？
如果模仿作者的幽默，可以做什么样的改写？
如果用{王尔德}的幽默风格对这个幽默进行改写是怎样的？
```

## 故事叙事

```Markdown
作者通过这本书呈现的故事内核，或者说最小故事是什么？
这个故事的「故事性」强吗？为什么？
这个故事的主人翁是谁？故事发生在什么时间、什么地点？先后经历了哪些事件？请以「事件一、事件二、事件三......事件n」进行表格呈现。
这个故事涉及人性的深层次问题吗？
假如要模仿作者写一个故事，你该如何写？
假如要模仿作者续写这个故事，你该如何写？
```

## 文采美感

```Markdown
现在你是一个文采大师，我了解到{XXX}在{XXX}中主要采用了{主流认知方式：文采美感}来阐述他的观点。当你阅读了他的这本书之后，请问：

作者给到了你怎样的意象和感觉？
你觉得本书写的最好的5个字词、句子、篇章是哪些？
作者的字词、句子、篇章有哪些可以学习和模仿的地方？
作者唤醒了你什么样的情绪与感知？
如果要寻找三位有着杰出作品的作家，你觉得应该是哪三位的文采风格与这位作者类似？
```

## 视觉美感

```Markdown
作者的作品向你展现了什么样的形象？
作者的作品有哪些独特的风格与美感？
模仿作者的设视觉设计，你可以应用到什么领域？
作者唤醒了你怎样的情绪与感知？
```
